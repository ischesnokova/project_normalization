## 2.Нормализация по рядам особенно важна для глобальных нейронных моделей, но влияет неоднозачно для моделей из разных классов

**Цель:** Проверка гипотезы "Нормализация по рядам особенно важна для глобальных нейронных моделей, но влияет неоднозначно для моделей из разных классов"

## Результат
Можно утверждать, что качество прогнозирования нейросетевых моделей
действительно крайне чувствительно к стандартизации/нормализации. 
Что касается не нейросетевых моделей, то на нашем примере модели градиентного
бустинга разница в метриках качества не является ярковыраженной
